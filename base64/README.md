# Base 64

## Uso:

- Para codificar un archivo usar:
``` bash
$python3 base64_c.py codificar <nombre archivo> <nombre archivo destino>
```

- Para decodificar un archivo usar:
``` bash
$python3 base64_c.py decodificar <nombre archivo> <nombre archivo destino>
```
